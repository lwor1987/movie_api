HTTP Method Request URL Request body data format
GET Get all movie lists "/movies" Return a list of All movies to the user
GET Get movie by title "/movies/:title" Return a list of one movie by title
GET Get movie by genre "/movies/genre/:title" Return data about genre (description) by name/title(e.g., "Thriller")
GET Get movie director name "/movies/director/:name" Return data about a director (bio, birth year, death year) by name
POST Allow new users to register "/users" A new user will be added using JSON format
PUT Allow users to update their user info(username) "/users/:username"
POST Allow users to add a movie to their list of favorites (showing only the text that a movie has been added-more on this later) "/users/:username/movies/:movieId"
DELETE Allow users to remove a movie from their list of favorites(showing only a text that a movie has been removed-more on this later) "/users/:username/movies/:movieId"
DELETE Allow existing users to deregister(showing only a text that a user email has been removed-more on this later) "/users"
