MOVIE API

This is a "movies" web application. The user will be able to access information about movies, directors, and genres on this application. The user will be able to sign up and make updates to the personal information and also their movie list.

Request Url Responses
GET /movies Return a list of All movies to the user.
GET /movies/:title Return date (description, genre,director,image URL,whetheer it's featured or not) about a single movie by title to the user.
GET /movies/:title/genre Return data about genre (description) by name/title(e.g., "Thriller")
GET /movies/director/:name Return data about a director (bio, birth year, death year) by name.
POST /newUser Allow new users to register.
POST /newUser/:id/favorites Allow users to add a movie to their list of favorites (showing only the text that a movie has been added-more on this later).
PUT /newUser/:id/info Allow users to update their user info(username).
DELETE /newUser/:/id/favorites Allow users to remove a movie from their list of favorites(showing only a text that a movie has been removed-more on this later).
DELETE /newUser Allowexisting users to deregister(showing only a text that a user email has been removed-more on this later).
