MOVIE API

This is a "movies" web application. The user will be able to access information about movies, directors, and genres on this application. The user will be able to sign up and make updates to the personal information and also their movie list.

Request Url Responses
GET / Homepage for myFlix App.
GET /documentation Get Documentation fo myFlix App.
GET /movies Return a list of All movies to the user.
GET /movies/:title Return date (description, genre,director,image URL,whetheer it's featured or not) about a single movie by title to the user.
GET /movies/:title/genre Return data about genre (description) by name/title(e.g., "Thriller")
GET /movies/director/:name Return data about a director (bio, birth year, death year) by name.
GET /users Get a list of all the users.
GET /users/:Username Get a user by username. POST /users Allow new users to register.
POST /users/:Username/movies/:MovieID Allow users to add a movie to their list of favorites (showing only the text that a movie has been added-more on this later).
PUT /users/:Username Allow users to update their user info(username).
DELETE /users/://username/movies/:MovieID Allow users to remove a movie from their list of favorites(showing only a text that a movie has been removed-more on this later).
DELETE /users/:Username Allow existing users to deregister(showing only a text that a user email has been removed-more on this later).
